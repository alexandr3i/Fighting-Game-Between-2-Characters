# Fighting Game Between 2 Characters (C++ OOP)

A **text-based console game** that simulates a battle between two characters: **the hero** and **the enemy**.  
Each character has at least one type of attack, and the player can also use a **consumable item** during the fight.

This project is written in **C++** and is structured as an **OOP (Object-Oriented Programming)** exercise.

---

## Repository contents
- `Fighting Game Between 2 Characters - OOP Project.cpp` - the C++ source code (main program)
- `Fighting Game Between 2 Characters - OOP Project.docx` - project documentation

---

## How to compile & run

### Windows (MinGW-w64)
1. Install **MinGW-w64** so `g++` is available in your PATH.
2. Open Command Prompt in the repo folder.
3. Compile:
   - g++ "Fighting Game Between 2 Characters - OOP Project.cpp" -o game.exe
4. Run:
   - game.exe

### Linux / macOS (GCC or Clang)
1. Open a terminal in the repo folder.
2. Compile:
   - g++ "Fighting Game Between 2 Characters - OOP Project.cpp" -o game
3. Run:
   - ./game
